DOM

DOM Gives us the ability to change elements on a a page, with Java Script the things you can change are.

  • All the HTML elements in the page
  • All the HTML attributes in the page
  • All the CSS styles in the page
  • Remove existing HTML elements and attributes
  • Add new HTML elements and attributes
  • React to all existing HTML events in the page
  • Create new HTML events in the page

We won't be going over all of these today but this here are a few examples

You can trigger DOM on events like button or mouse over
