Data Types

Variables are containers that you store things in There are several types of date taht ca be stored the main three are Number, String, and Boolean. The number type can store well numbers. The string data type holds strings of text. The boolean type hold only one of two values True, and False. 


        var numberDataType = 1;
        var booleanDataType = True;
        var stringDataType = "I'm a string";

You can even do math with variables and the number data type


        var a = 5;
        var b = 10;
        var c = a * c;
        var e = 10 / 2;
