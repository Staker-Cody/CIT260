CSS Transitions

CSS transitions give you the ability to change the styling of something when the user interacts with in my example below I increase the size of the photos when the user hovers over them. There are several different attributes you can change you can also change the speed and how it looks as it transitions 


             .tone {
            width: 100px;
            height: 100px;
            -webkit-transition: width 2s, height 3s; /* Safari */
            transition: width 1s, height 1s;



        }
        .tone:hover {

            width: 300px;
            height: 300px;
        }
